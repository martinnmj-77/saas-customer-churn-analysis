# 📊 SaaS Customer Churn & Health Analysis

## 📌 Problem Statement
Customer churn is a critical challenge for SaaS businesses, directly impacting revenue and growth.  
This project aims to analyze customer behavior, identify key churn drivers, and develop a customer health framework to support data-driven retention strategies.

---

## 📊 Dataset
- Telco Customer Churn Dataset (Kaggle)  
- ~7,000 customers with demographic, subscription, and churn data  

---

## ⚙️ Tools & Technologies
- **SQL** – Data querying and segmentation  
- **Python (Pandas, NumPy, Scikit-learn)** – Data cleaning, analysis, and modeling  
- **Tableau** – Interactive dashboard and data visualization  

---

## 🔍 Approach

### 1. Data Cleaning
- Handled missing values and inconsistent data types  
- Converted numerical fields and removed anomalies  
- Prepared structured dataset for analysis  

---

### 2. Exploratory Data Analysis (EDA)
- Analyzed churn distribution across customer segments  
- Identified patterns based on tenure, contract type, and pricing  
- Explored relationships between customer behavior and churn  

---

### 3. SQL Analysis
- Calculated churn rates across multiple dimensions  
- Identified high-risk customer segments  
- Performed aggregation and segmentation using conditional logic  

---

### 4. Dashboard Development (Tableau)
- Built an interactive dashboard to visualize churn drivers  
- Included KPIs, segmentation charts, and churn heatmaps  
- Enabled filtering by customer attributes for deeper insights  

---

### 5. Customer Health Score
- Developed a rule-based scoring system using:
  - Customer tenure  
  - Contract type  
  - Monthly charges  
- Segmented customers into:
  - 🟢 Healthy  
  - 🟡 At Risk  
  - 🔴 High Risk  

---

## 💡 Key Insights
- Customers on **month-to-month contracts** show significantly higher churn rates  
- Customers with **low tenure (< 6 months)** are most likely to churn  
- Higher **monthly charges** are associated with increased churn probability  

---

## 🚀 Business Recommendations
- Introduce incentives for long-term contracts to improve retention  
- Strengthen onboarding experience for new customers  
- Target high-risk segments with personalized retention strategies
